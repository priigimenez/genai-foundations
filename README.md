## Objetivo
Explorar fundamentos de LLMs y GenAI con foco en:
- uso responsable de datos
- calidad
- trazabilidad
- posibles aplicaciones en Data Governance

## Contenidos
- Conceptos básicos de LLMs
- Ejemplos de prompts
- Primeras llamadas a APIs
- Reflexiones sobre riesgos y control del dato

## Relación con Data Governance
Este repositorio busca entender cómo los modelos de lenguaje interactúan con datos y qué desafíos de gobierno surgen.
